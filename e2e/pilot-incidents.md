# ConceptManager — Journal d'incidents pilote

## Classification

| Niveau | Definition | Action |
|--------|-----------|--------|
| **BLOQUANT** | Fonction inutilisable, perte de donnees, crash | Correction immediate, re-test |
| **CRITIQUE** | Fonction degradee, workaround possible | Correction sous 48h |
| **OPTIMISATION** | Amelioration UX, performance | Backlog post-pilote |
| **NICE-TO-HAVE** | Suggestion, idee | Backlog V2 |

**Regle pilote** : On corrige uniquement BLOQUANT + regressions. Le reste = backlog.
