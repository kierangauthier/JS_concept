## Parcours type d'un RDV (~45 min)

```
0:00 — Accueil (5 min)
        Présentation rapide, brise-glace.

5:00 — Pitch Slides 1-3 (10 min)
        Contexte → Problème → Solution.
        Cf. pitch-deck-JS.md slides 1, 2, 3.

15:00 — DÉMO LIVE (15 min)
        Suivre demo-script-JS.md MINUTÉ.
        Moment clé : export Factur-X à 8:00.

30:00 — Slide prix + Q&A (10 min)
        Cf. pitch-deck-JS.md slide 4.
        Anticiper les objections (objections-playbook.md).

40:00 — Clôture (5 min)
        Question magique : « Qu'est-ce qui vous empêche
        de signer cette semaine ? »
        Si signe → contrat-type.md + CGV.md.
        Si pas signe → RDV de suivi daté.
```

## Avant chaque RDV — checklist

### La veille
- [ ] Tester la démo complète en conditions réelles (même compte, mêmes données).
- [ ] Vérifier que le Factur-X se télécharge et s'ouvre (`stage3-factur-x.pdf` comme référence).
- [ ] Préparer un jeu de données propre et crédible dans l'outil.
- [ ] Relire les 10 objections dans `objections-playbook.md`.
- [ ] Imprimer 2 exemplaires du contrat + CGV (ou activer DocuSign).
- [ ] Préparer le devis personnalisé au nom du client.

### Le jour-même
- [ ] Laptop chargé + chargeur + 4G de secours.
- [ ] Cartes de visite.
- [ ] Logo du client pré-chargé dans l'outil pour la démo.
- [ ] Bloc-notes pour noter les objections et demandes.
- [ ] Notifications coupées (Slack, Teams, email) pendant la démo.

## Principes à garder en tête

1. **Ne jamais parler d'Ostara.** C'est l'outil interne. Le client achète « ConceptManager », point.
2. **Ne jamais baisser le mensuel sans remonter autre chose.** Le mensuel est l'argent récurrent.
3. **Ne jamais laisser partir sur "je réfléchis".** Creuser l'objection.
4. **Ne jamais promettre ce qui n'existe pas.** "Je note et je reviens vers vous" au pire.
5. **Conclure le RDV avec une prochaine étape datée.** Toujours.

## En cas de signature

1. Signer le contrat + CGV (DocuSign idéalement).
2. Facture Setup 50 % émise le jour-même.
3. Email de bienvenue : coordonnées, prochaine étape, documents à fournir.
4. Date de kick-off projet planifiée sous 7 jours.
5. **Demander au client un témoignage après 60 jours d'utilisation** (à verrouiller à la signature, pas après).
